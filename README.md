## 🚀 Instalación

```bash
git clone https://github.com/usuario/pong.git

cd proyecto

```
Si utilizas gradle con eso es suficiente.

Si utilizas maven necesitas convertir el archivo gradle.build en pom.xml con alguna herramienta .

---

## ▶️ Uso

### *Linux:*

Para ejecutar el programa situate en la carpeta raiz del proyecto con:

```bash
cd nombre-del-directorio/
```

y ejecuta en terminal:

```bash
gradle fatJar
```

```bash
java -jar /app/build/libs/app-1.0.jar
```

### *Windows:*
Puedes ejecutarlo siguiendo los mismos pasos que linux o con el ```exe``` en la carpeta ```dist```
